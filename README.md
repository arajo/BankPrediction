# 입금기관 추천 모델

계좌번호 입력 기반 입금기관(은행/증권사) 추천 모델입니다.  
대상 입금기관은 총 52개(은행: 25, 증권사: 27)입니다.

### Model
- Multi-class classification with LSTM
- (Version 5) Test Data Accuracy: 91.35%
- (Version 3) Test Data Accuracy: 89.81%

### Version
- python 3.10.10
